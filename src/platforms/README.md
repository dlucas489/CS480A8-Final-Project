# src/platforms/

Published hardware parameters for each quantum computing platform.

Planned modules:
- `specs.py` — dictionary / dataclass of T₁ (µs) and t_gate (ns) per platform
- `ibm.py`   — IBM Quantum (Eagle r3 / Heron) parameters and citations
- `google.py` — Google Quantum AI (Sycamore) parameters and citations
- `intel.py`  — Intel Tunnel Falls (silicon spin) parameters and citations
- `ionq.py`   — IonQ Aria (trapped ion) parameters and citations
