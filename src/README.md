# src/

Source code for the Grover amplitude-damping simulation.

## Submodules

| Directory | Purpose |
|---|---|
| `grover/` | Grover circuit construction (oracle, diffuser, full circuit) |
| `noise/` | Amplitude damping channel helpers and γ derivation |
| `platforms/` | Published T₁ / t_gate hardware specs per platform |
| `analysis/` | Success probability computation, threshold detection, plotting |
